**浏览**（Explore）页面是你查看和搜索 Frigate 已保存的每个**追踪目标**的地方。默认情况下，它会按标签对最近的追踪目标进行分组；当启用了[语义搜索](/configuration/semantic_search)时，你还可以通过自然语言描述或视觉相似度进行搜索。选择任意目标可打开详情面板，查看其快照、生命周期和元数据。

本页介绍如何_使用_浏览视图。有关底层功能的_配置_方式，请参阅[语义搜索](/configuration/semantic_search)和[生成式 AI 描述](/configuration/genai/genai_objects)。

## 浏览追踪目标

默认视图按标签将最近的追踪目标分组为行显示——_人_、_车_、_狗_ 等——每行标注有目标类型和数量。行末的箭头可打开该标签的完整可筛选网格。

点击缩略图可打开其[详情对话框](#追踪目标详情)；右键点击或长按缩略图可打开[操作菜单](#操作与批量选择)。你可以在视图设置中切换到更紧凑的网格布局，并调整列数。

## 搜索

当启用[语义搜索](/configuration/semantic_search)后，会出现一个搜索栏，将两种功能合并在一个输入框中：

- **自然语言搜索**——输入自由文本查询并按回车，对追踪目标进行语义搜索。
- **筛选标记**——输入 `key:` 获取建议，再输入值，即可添加结构化筛选器。每个筛选器会变成一个可移除的标签，你可以将多个筛选器串联使用。

你可以使用星标图标保存搜索以便日后重新加载，使用清除搜索图标清除所有内容。帮助弹窗会说明标记语法，例如：

```
cameras:front_door label:person before:01012024 time_range:3:00PM-4:00PM
```

### 筛选器参考

最常用的筛选标记如下：

| 筛选器                    | 说明                                                                   |
| ------------------------- | ---------------------------------------------------------------------- |
| **摄像头**                | 限制为一个或多个摄像头。                                               |
| **标签**                  | 目标标签（person、car 等）。                                           |
| **子标签**                | 识别的子标签（如识别的人脸或名称）。                                   |
| **属性**                  | 应用于目标的分类属性。                                                 |
| **识别的车牌**            | 匹配已识别的车牌。                                                     |
| **区域**                  | 进入特定区域的目标。                                                   |
| **之前 / 之后**           | 限制日期范围。                                                         |
| **时间范围**              | 限制一天中的某个时间段（`HH:MM-HH:MM`）。                              |
| **最低 / 最高分数**       | 按目标的置信度分数筛选。                                               |
| **最低 / 最高速度**       | 按估计速度筛选（需配置速度估算）。                                     |
| **包含快照 / 包含片段**   | 仅显示保存了快照或录像的目标。                                         |
| **已提交到 Frigate+**     | 仅显示已提交的目标（需启用 Frigate+）。                                |
| **搜索类型**              | 语义搜索是匹配目标的**缩略图**还是其**描述**。                         |

### 排序

当筛选或搜索处于活动状态时，**排序**控件可让你按**日期**、**目标分数**或**估计速度**（升序或降序）排列结果。当语义查询或相似度搜索处于活动状态时，结果还可以按**相关度**排序。

### 缩略图和描述搜索

- **搜索类型**设置控制文本查询是匹配每个目标的**缩略图**还是其**描述**。每个结果会显示匹配的是哪一种以及置信度。

自然语言搜索、缩略图搜索和描述搜索都需要启用[语义搜索](/configuration/semantic_search)。

## 追踪目标详情

选择一个目标会打开**追踪目标详情**对话框。使用箭头（或左右方向键）切换到上一个或下一个目标。对话框有两个选项卡：

- **快照**或**缩略图**——保存的快照（或缩略图）。
- **追踪详情**——目标的生命周期，当目标有录像时可用。它列出了每个重要时刻（检测到、进入区域、变为活动或静止状态、离开等）；点击某个时刻会播放该段录像并叠加边界框。设置弹窗可让你显示所有区域并调整标注偏移量。

详情面板显示目标的**标签**、**分数**、**摄像头**、**时间戳**、估计的**速度**、任何**识别的车牌**和**分类属性**，以及其**描述**。管理员可以内联编辑子标签、车牌和属性。

**描述**可以手动编辑，并且——当启用了[生成式 AI 描述](/configuration/genai/genai_objects)且目标的生命周期已结束——可以从快照或缩略图重新生成。对于 `speech` 目标，当启用音频转录时可使用**转录**操作。当启用 [Frigate+](/integrations/plus) 时，管理员可以直接从此面板提交快照以改进其模型。

## 操作与批量选择

右键点击或长按一个目标（在网格或其缩略图中）会打开操作菜单，选项包括**下载**视频、快照或纯净快照；**查看追踪详情**；**查找相似**；**添加触发器**；**在历史记录中查看**；以及**删除追踪目标**。

:::note

删除追踪目标会移除其快照、嵌入向量和追踪详情条目，但该目标在[历史记录](/usage/history)中的录像**不会**被删除。

:::

要对多个目标进行操作，请 Ctrl/Cmd 点击或右键点击以开始选择（选中的图块会出现蓝色边框），然后使用工具栏全选、清除选择或删除（管理员）。

## 语义搜索 - 使用与最佳实践 {#usage-and-best-practices}

1. 语义搜索与浏览页面上的其他筛选器配合使用。将传统筛选与语义搜索结合使用可获得最佳效果。
2. 搜索场景中的特定目标时使用缩略图搜索类型。试图辨别目标意图时使用描述搜索类型。
3. 由于 Frigate 使用的 AI 模型的训练方式，文本与图像嵌入距离的比较通常意味着在多模态（`缩略图`和`描述`）搜索中，匹配`描述`的结果会优先出现，即使`缩略图`嵌入可能是更好的匹配。尝试调整"搜索类型"设置以帮助找到你要找的内容。注意，如果你只为特定目标或区域生成描述，这可能导致搜索结果优先显示有描述的目标，即使没有描述的目标更相关。
4. 使你的搜索语言和语调与你正在寻找的内容紧密匹配。如果你使用缩略图搜索，**将查询表述为图像标题**。搜索"红色汽车"可能不如"红色轿车在阳光明媚的一天行驶在住宅区街道上"效果好。
5. 缩略图上的语义搜索在匹配占据大部分画面的较大目标时往往返回更好的结果。像"猫"这样的小目标通常效果不佳。
6. 多尝试！找一个你想测试的追踪目标，开始输入关键词和短语，看看什么对你有效。

## 触发器

从目标的操作菜单中，**添加触发器**可设置一个按摄像头的触发器，当类似目标出现时使用语义搜索自动执行操作（通知、子标签或属性）。触发器需要语义搜索，并在设置 > 增强功能 > 触发器下管理。有关完整配置和最佳实践，请参阅[触发器](/configuration/semantic_search#triggers)。
